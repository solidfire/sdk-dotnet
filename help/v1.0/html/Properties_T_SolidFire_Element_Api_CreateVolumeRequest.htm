List of Name/Value pairs in JSON object format.

Should the volume provides 512-byte sector emulation?
Name
Name of the volume. Not required to be unique, but it is recommended. May be 1 to 64 characters in length.
Qos
Initial quality of service settings for this volume.

Volumes created without specified QoS values are created with the default values for QoS. Default values for a volume can be found by running the GetDefaultQoS method.
TotalSize
Total size of the volume, in bytes. Size is rounded up to the nearest 1MB size.
