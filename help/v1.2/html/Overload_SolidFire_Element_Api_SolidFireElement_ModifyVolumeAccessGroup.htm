ModifyVolumeAccessGroup(ModifyVolumeAccessGroupRequest)
Update initiators and add or remove volumes from a volume access group. A specified initiator or volume that duplicates an existing volume or initiator in a volume access group is left as-is. If a value is not specified for volumes or initiators, the current list of initiators and volumes are not changed.

Often, it is easier to use the convenience functions to modify initiators and volumes independently:

AddInitiatorsToVolumeAccessGroup
RemoveInitiatorsFromVolumeAccessGroup
AddVolumesToVolumeAccessGroup
RemoveVolumesFromVolumeAccessGroup
