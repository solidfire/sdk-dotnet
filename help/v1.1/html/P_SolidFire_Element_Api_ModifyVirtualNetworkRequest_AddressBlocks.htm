ModifyVirtualNetworkRequestAddressBlocks Property
New addressBlock to set for this Virtual Network object. This may contain new address blocks to add to the existing object or it may omit unused address blocks that need to be removed. Alternatively, existing address blocks may be extended or reduced in size. The size of the starting addressBlocks for a Virtual Network object can only be increased, and can never be decreased. Attributes for this parameter are:
start: start of the IP address range. (String)
size: numbre of IP addresses to include in the block. (Integer)
